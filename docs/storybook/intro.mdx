# UTC Website Design System

Welcome to the Urban Tech Creative design system and component library.

## What is this?

This Storybook shows **components and experiments** for the UTC website. Use it to:

- Preview UI components in isolation
- Explore design experiments (especially Cube faces)
- Reference styling, primitives, and patterns

## How to navigate

**Sidebar** — Tap the menu icon (☰) on mobile to open it. Browse by:

- **Atoms / Molecules / Organisms** — Core design system components
- **Experiments** — Work in progress (Cube faces, buttons, layouts, etc.)
- **Cube/Face Primitives** — Building blocks for Cube face layouts (Cell, ImageBlock, TextBlock, etc.)

**Experiments** — Numbered stories (e.g. "7 – Portraits + Icon Quad") are prototypes; reference them by number in discussions.

## Quick links

Use the sidebar to jump to:

- **Cube Faces** — XR, Work, Collaborators, News
- **Cube/Face Primitives** — Cell, ImageBlock, TextBlock, IconQuad, and more
- **Atoms** — Frame, Accent, Heading, Icon

## On mobile

The sidebar is collapsed by default. Tap the menu icon in the top-left to open it and browse stories.
